Import parties from CSV
Upload a CSV file to bulk-add parties
Required CSV format:
Party Name, City, State, Salesperson Name, Grade
Example row:
Sharma Traders, Surat, Gujarat, Raunak, A

• First row must be the header row (as shown above)
• Salesperson name must exactly match a name in your system
• Grade is optional — use A, B, or C (leave blank to skip)
• CSV-imported parties are automatically marked as Existing clients
• Duplicates (same name + city) will be skipped
Import pricelist from CSV
Upload a CSV file to bulk-add products
Required CSV format:
Code, Name, Category, Unit, Price
Example row:
VT001, Classic Vest, Vest, Dozen, 240

• First row must be the header row
• Category must be one of: Vest, Brief, Trunk, Boxer, Thermal, Socks, Bra, Other
• Unit must be one of: Piece, Dozen, Box (10pcs), Set, Kg, Metre
• Price must be a number (no ₹ symbol)
• Duplicate codes will be skipped
